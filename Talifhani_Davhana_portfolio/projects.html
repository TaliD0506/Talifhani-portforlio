Academic Project E-commerce Web Application Full-Stack Development

Full-Stack Management & E-commerce Platform for a dress rental boutique

Web-based platform combining e-commerce, payment tracking, and admin dashboard management

Project Overview

I developed for Ozyde Dress Rental Boutique as a full-stack academic project, this platform supports an e-commerce dress rental management through a centralised admin dashboard. The system enables administrators to manage products and rental inventory, track customer payments and orders, and monitor business performance using structured data and visual statistics. While customers can browse the catalogue, book rentals, and track their orders through a user-friendly interface. Secure role-based authentication ensures admin-only access to sensitive operations, while Google Maps API integration allows customers to select accurate delivery and collection locations. The platform was designed with a modular PHP architecture to ensure scalability, maintainability, and a clean user experience.

Academic Project E-commerce

Lushaka Urithi – C2C E-commerce Platform

Culture-driven consumer-to-consumer marketplace celebrating African heritage

Project Overview

Lushaka Urithi is a full-stack C2C e-commerce platform developed as a final-year academic project. The platform enables users across South Africa to buy and sell traditional and pre-loved African clothing and accessories. It was designed to preserve cultural heritage while empowering local sellers through digital commerce.

The system includes buyer, seller, and admin dashboards, complete product and order management, cultural category filtering, and secure authentication. The project focuses on usability, inclusivity, and scalable architecture.
